# LINE 規約メモ

## ビジネスマネージャー利用規約

- 規約本文: <https://terms2.line.me/businessmanager?lang=ja>

### 第4条（禁止事項）

**1項 — 異なる法人アカウント間のリソース共有禁止

契約主体（法人）が異なる広告主アカウント（LINE 広告・LINE 公式アカウント等）を同一のビジネスマネージャー組織に接続し、オーディエンスや LINE Tag などのリソースを共有することは禁止。

同一法人内での複数プロバイダー・チャネルのまとめ管理はOK。

```txt
同一法人の LINE ビジネス ID
└── ビジネスマネージャー組織
    ├── プロバイダー A
    │   └── チャネル 1（Messaging API）
    └── プロバイダー B
        └── チャネル 3（Messaging API）
← これは同一法人なので問題なし
```

2項 — 他社データを使った配信禁止

自分の広告主アカウントにおいて、他の広告主のデータを使って広告配信やメッセージ配信を行うことは禁止。データの横流し・使い回し禁止。
