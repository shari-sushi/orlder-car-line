# LINE Messaging API でできること

## メッセージ系

| 機能                 | 概要                                                                                                                        |
| -------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| リッチメニュー       | 画面下部に固定のメニューUI（ボタン・画像付き）。画像は1枚で全体を表現し、**1MB以下・JPEG推奨**（PNGは圧縮しても超えやすい） |
| フレックスメッセージ | カード型UIで画像・ボタン・テキストを自由レイアウト                                                                          |
| クイックリプライ     | 返信候補をボタンで提示                                                                                                      |
| ボタンテンプレート   | 選択肢ボタン付きメッセージ                                                                                                  |
| カルーセル           | 横スクロールできる複数カード                                                                                                |

## インタラクション系

| 機能               | 概要                                                          |
| ------------------ | ------------------------------------------------------------- |
| Postback           | ボタン押下で任意データをサーバーに送信（URLには表示されない） |
| 画像メッセージ     | 画像を受け取って解析など                                      |
| 位置情報メッセージ | ユーザーの現在地を受け取る                                    |

## ユーザー管理系

| 機能                      | 概要                             |
| ------------------------- | -------------------------------- |
| フォロー/ブロックイベント | 友達追加・解除のタイミングで処理 |
| プロフィール取得          | ユーザー名・アイコン取得         |
| グループ/ルーム           | グループチャットでの動作         |

# このアプリへの追加案

現状の「車検索URLを返す」機能をベースに拡張するアイデア。

| 優先度 | 機能                 | 概要                                                       |
| ------ | -------------------- | ---------------------------------------------------------- |
| 高     | クイックリプライ     | 「再検索」「条件変更」などのボタンを返信に添付             |
| 高     | フレックスメッセージ | 検索URLをリンクボタン付きカードで返す（見栄えUP）          |
| 中     | リッチメニュー       | 「使い方」「検索する」などを常時表示                       |
| 中     | Postback             | 年式・距離の選択肢をボタンで選ばせる（入力の手間を減らす） |
| 低     | お気に入り保存       | Redisに検索条件を保存して「前回の条件で検索」              |
