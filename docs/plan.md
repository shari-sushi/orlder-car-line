# 実装計画

## 背景

LINE bot（Go）で中古車カルーセルを返す機能を実装中。

- `SUV` と送信 → カーセンサーから **ライズ・XV・ジムニー** の最新1台を並列スクレイピングしてカルーセル表示
- 各カードに **実際の画像・グレード・価格・年式・走行距離** を表示
- グレードは img の alt 属性から、価格/年式/距離は HTML 内の regex で抽出
- detail URL を中心に前後 4000 文字のブロックで絞り込む方式

---
