---

以下の手順で計画書を着手状態にし、実装を開始してください。

## 対象ファイル

$ARGUMENTS が指定されていればそのパス、なければ現在IDEで開いている計画書ファイルを対象にする。

## 手順

### 1. 計画書を読む

対象の計画書ファイルを読み、実装内容を把握する。

### 2. wip に移動

```bash
git mv VIBES/1_plan/<ファイル名>.md VIBES/2_wip/<ファイル名>.md
```

### 3. 進捗セクションを追加

ファイル末尾に以下を追記する（`---` の前に空行を入れること）。
計画書の「実装計画」セクションをもとに、チェックボックス形式でタスクを列挙する。
タスクの粒度は実装計画の内容に合わせる。

```markdown
---

## 進捗

- [ ] タスク1
- [ ] タスク2

### メモ・課題

-（実装中に気づいた点をここに書く）
```

### 4. 実装開始

`feedback/lessons.md` を確認し、関連するパターンを適用した上で実装に入る。
